## 💡 Study Tips

1. **Open DevTools Console** — Test every code snippet there.
2. **Type, don't copy** — Muscle memory matters in programming.
3. **Build mini-projects** — After each chapter, make something small.
4. **Use `console.log()` everywhere** — It's your best debugging tool.
5. **Read error messages** — They tell you exactly what went wrong and where.
6. **MDN is your bible** — [developer.mozilla.org/en-US/docs/Web/JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript)
